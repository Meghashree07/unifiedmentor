## Description  
This is an interactive quiz web app built using HTML, CSS, and JavaScript.  
It provides a user-friendly interface where users can answer quiz questions, and get immediate feedback with scores.  
The goal of this project is to practice DOM manipulation, event handling, and build a simple but functional web application as part of my learning journey.

## Features  
- Multiple-choice quiz questions  
- Immediate result evaluation after submitting answers  
- Simple and clean UI  
- Works on any modern browser
